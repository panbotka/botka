### What to Create

1. **`internal/handlers/tag.go`**:
   - `GET /api/v1/tags` — List all tags
   - `POST /api/v1/tags` — Create tag (name, color)
   - `PUT /api/v1/tags/:id` — Update tag
   - `DELETE /api/v1/tags/:id` — Delete tag
   - `GET /api/v1/tags/:id/threads/count` — Count threads with this tag

2. **`internal/handlers/persona.go`**:
   - `GET /api/v1/personas` — List personas ordered by sort_order
   - `POST /api/v1/personas` — Create persona (name, system_prompt, default_model, icon, starter_message, sort_order)
   - `PUT /api/v1/personas/:id` — Update persona
   - `DELETE /api/v1/personas/:id` — Delete persona

3. **`internal/handlers/memory.go`**:
   - `GET /api/v1/memories` — List all memories
   - `POST /api/v1/memories` — Create memory (content)
   - `PUT /api/v1/memories/:id` — Update memory
   - `DELETE /api/v1/memories/:id` — Delete memory

4. **`internal/handlers/search.go`**:
   - `GET /api/v1/search?q=query` — Full-text search across messages using PostgreSQL tsvector
   - Returns matching message snippets with thread info
   - Use `db.Raw()` with `plainto_tsquery` and `ts_headline` for highlighted snippets

5. **`internal/handlers/transcribe.go`**:
   - `GET /api/v1/transcribe/status` — Check if whisper is available (ping OpenClaw)
   - `POST /api/v1/transcribe` — Proxy audio to OpenClaw's whisper endpoint, return transcription text
   - Multipart form data handling for audio file

6. **`internal/handlers/process.go`**:
   - `GET /api/v1/processes` — List active Claude Code chat processes from claude.Registry
   - `DELETE /api/v1/processes/:id` — Kill a process by thread ID

7. **`internal/handlers/status.go`** (or add to existing file):
   - `GET /api/v1/models` — Return list of available models from config
   - `GET /api/v1/status` — Return server status (default model, whisper enabled, etc.)

8. **Wire all routes** in main.go setupRouter.

### GORM Conversion Notes
All these are straightforward GORM CRUD:
- Tags: `db.Find(&tags)`, `db.Create(&tag)`, etc.
- Personas: same pattern
- Memories: same pattern, UUID primary key
- Search: use `db.Raw()` for tsvector full-text search query
- Thread tags: `db.Model(&thread).Association("Tags").Replace(tags)`
