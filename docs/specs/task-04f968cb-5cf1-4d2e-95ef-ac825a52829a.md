# Sjednotit formát data a času na DD/MM/YYYY HH:MM

Všude kde se zobrazuje konkrétní datum nebo čas, použít formát `DD/MM/YYYY` (např. `21/04/2026`) a `HH:MM` (např. `16:21`). Relativní časy ("2m ago", "3h ago", "Yesterday" apod.) ponechat beze změny.

## Requirements

### Frontend

- Vytvořit sdílené utility funkce ve `frontend/src/utils/` (nový soubor nebo rozšířit existující):
  - `formatDate(date)` → `DD/MM/YYYY` (např. `21/04/2026`)
  - `formatTime(date)` → `HH:MM` (např. `16:21`)
  - `formatDateTime(date)` → `DD/MM/YYYY HH:MM` (např. `21/04/2026 16:21`)

- **MessageBubble.tsx** (řádky 35-48): Funkce `formatTimestamp()` — změnit formát. Pokud je dnes, zobrazit `HH:MM`. Pokud jiný den, zobrazit `DD/MM/YYYY HH:MM`. Logika "dnes vs. jiný den" zůstává.

- **TaskDetailPage.tsx** (řádky 57-59): Nahradit `toLocaleString()` za `formatDateTime()` pro Created, Updated, Started, Finished timestamps.

- **SettingsPage.tsx** (řádek 1211): Nahradit `toLocaleDateString()` za `formatDate()` pro datum vytvoření passkey.

- **exportThread.ts** (řádky 5-12): Nahradit `toLocaleString('en-GB', ...)` za `formatDateTime()`.

- **ThreadSidebar.tsx** (řádky 218-231): Relativní časy (now, Xm, Xh, Yesterday, weekday) ponechat. Pouze fallback pro starší data (>= 7 dní) změnit z `"31 Mar"` na `DD/MM/YYYY`.

- **TaskList.tsx** (řádky 139-151): Relativní časy (just now, Xm ago, Xh ago, Xd ago) ponechat. Pouze fallback pro starší data (>= 7 dní) změnit z `toLocaleDateString()` na `DD/MM/YYYY`.

- **ProjectDetailPage.tsx** (řádky 91-102): Relativní časy ponechat beze změny.

- **DashboardPage.tsx** (řádky 49-56): Countdown timer ponechat beze změny.

### Backend

- **internal/mcp/tools.go** (řádek 15): Změnit konstantu `timeFmt` z `"2006-01-02 15:04"` na `"02/01/2006 15:04"` (Go reference time layout pro DD/MM/YYYY HH:MM).

- **internal/mcp/tools_thread.go**: Používá stejnou konstantu `timeFmt`, ověřit že se změna propaguje.

### Co NEMĚNIT

- ISO 8601 / RFC3339 formáty v API responsech (backend JSON serialization)
- ISO 8601 při vytváření zpráv (`ChatView.tsx`, `SSEContext.tsx`)
- Relativní časy všude kde jsou
- Countdown timer na DashboardPage
- Number formatting (`toLocaleString()` pro tokeny apod.)

## Acceptance Criteria

- Všechna konkrétní data v UI se zobrazují jako DD/MM/YYYY
- Všechny konkrétní časy v UI se zobrazují jako HH:MM
- MCP tool outputy používají formát DD/MM/YYYY HH:MM
- Relativní časy fungují stejně jako předtím
- `make check` prochází
- Existující testy aktualizovány na nový formát
