### Design System Translation Guide

| Chatovadlo (dark) | Botka (Saiduler-style) |
|---|---|
| bg-gray-950 | bg-white or bg-zinc-50 |
| bg-gray-900 | bg-zinc-100 |
| bg-gray-800 | bg-zinc-200 |
| text-gray-100 | text-zinc-900 |
| text-gray-400 | text-zinc-500 |
| border-gray-700 | border-zinc-200 |
| hover:bg-gray-800 | hover:bg-zinc-100 |
| accent colors | Keep emerald for success, red for error, amber for warning, blue for info |

Also support dark mode via a `.dark` class or `prefers-color-scheme` media query — keep the original dark colors accessible when dark mode is toggled in settings.
