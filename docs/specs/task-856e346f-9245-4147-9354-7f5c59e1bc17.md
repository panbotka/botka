## Requirements

On the dashboard in the active sessions section (active chat sessions):
1. **Click to open chat** — clicking a session box should navigate to the corresponding chat thread.
2. **Slightly enlarge boxes** — session boxes are a bit small, increase their size slightly for better readability and clickability.

## Implementation

1. **Frontend:** Find the component displaying active sessions on the dashboard (likely in `frontend/src/pages/` or `frontend/src/components/`). Search for "session", "active", "registry" in components.
2. **Link to chat:** Each session should have a `thread_id` — use it for navigation to `/chat/{thread_id}` or the equivalent route. Wrap the box in a `<Link>` or add an `onClick` with `navigate()`.
3. **Enlarge boxes:** Slightly increase padding, font size, or overall box dimensions. Not a dramatic change, just a bit larger for better UX.

## Where to look

- Dashboard page: `frontend/src/pages/`
- Session/registry components: `frontend/src/components/`
- Backend endpoint for sessions: `internal/handlers/` — search for "session", "registry"
- Router/routes: `frontend/src/` — verify that a route exists for chat thread detail
